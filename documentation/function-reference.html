Function reference

cajal ( elementId )

Constructor for a new cajal instance

Parameter

  • elementId: id of the canvas element that should be drawn on

Return value

  • the cajal instance

cajal.addItem ( itemId, item )

Adds an item to the item array

Parameter

  • itemId: Name of the item (unique value)
  • item: item object

Return value

  • cajal instance or false if itemId already exists or param item is no object

cajal.getItem ( itemId )

Get the item with specific itemId or false

Parameter

  • itemId: item id of the specific item

Return value

  • the item with specific itemId or false

cajal.setItem ( itemId, item )

Override item with specific itemId

Parameter

  • itemId: item id of specific item
  • item: new item object

Return value

  • cajal instance or false if itemId does not exist

cajal.deleteItem ( itemId )

Delete item from item array

Parameter

  • itemId: item id of the item

Return value

  • cajal instance on success or false if item does not exist

cajal.getItemPosition ( itemId )

Get the key of the item in the item array

Parameter

  • itemId: item id of the item

Return value

  • key of the item or false if item does not exist

cajal.up ( itemId )

Move item one layer up

Parameter

  • itemId: item id of the item

Return value

  • cajal instance on success or false

cajal.down ( itemId )

Move item one layer down

Parameter

  • itemId: item id of the item

Return value

  • cajal instance on success or false

cajal.top ( itemId )

Move item on the top

Parameter

  • itemId: item id of the item

Return value

  • cajal instance on success or false

cajal.bottom ( itemId )

Move item to the bottom

Parameter

  • itemId: item id of the item

Return value

  • cajal instance on success or false

cajal.draw ( options )

Draw all Objects stored in the item array to the canvas

The canvas will be cleared if the flag "autoClearCanvas" is true

Parameter

  • options: literal object with draw options that will override the draw options of each item for this draw call

Return value

  • cajal instance

cajal.clear ( )

Clears the complete canvas

Return value

  • cajal instance

cajal.trigger ( animation,duration )

Triggers the given animation with specific duration

Parameter

  • animation: callback function
  • duration: duration of the animation in frames

Return value

  • cajal instance

cajal.stop ( animation )

Stops animation

Parameter

  • animation: callback function

Return value

  • cajal instance or false if animation does not exist in loop

cajal.loopAnimationExists ( animation )

Checks weather an animation already exists in the animation array

Parameter

  • animation: callback function

Return value

  • true on success or false

cajal.loopAddAnimation ( animation, duration )

Add animation to the animation loop and start the loop if it is not started yet

Parameter

  • animation: callback function
  • duration: duration for this animation

cajal.loop ( )

Loop routine that calls each animation callback, increments the frame number,

removes animations that reached their duration limit, and stops the loop

interval if no animations are left in the loop

cajal.LinearGradient ( x1, y1, x2, y2 )

Create a linear gradient

color stops can be added using the colorStop method

Parameter

  • x1: x coordinate for the gradient start
  • y1: y coordinate for the gradient start
  • x2: x coordinate for the gradient end
  • y2: y coordinate for the gradient end

Return value

  • gradient object

cajal.RadialGradient ( x1, y1, r1, x2, y2, r2 )

Create a radial gradient

color stops can be added using the colorStop method

Parameter

  • x1: x coordinate for the gradient start
  • y1: y coordinate for the gradient start
  • r1: radius for the gradient start
  • x2: x coordinate for the gradient end
  • y2: y coordinate for the gradient end
  • r2: radius for the gradient end

Return value

  • gradient object

gradient.colorStop ( pos, color )

add a color stop to the gradient

Parameter

  • pos: float number between 0 and 1 where the color is located on the gradient
  • color: hex or rgb(a) value of the color

Return value

  • gradient instance

gradient.draw ( context )

Get the canvas gradient object for drawing the gradient

Parameter

  • context: 2D rendering context

Return value

  • the canvas gradient object

item.clone ( )

Clone the item

Return value

  • clone of the item

item.hide ( )

Hide item

Hidden items are not drawn to the canvas

Return value

  • item instance

item.show ( )

Show item if hidden

Return value

  • item instance

item.changeMatrix ( m11, m12, m21, m22, dx, dy )

Change the matrix of the canvas element for this item by multiplication with current matrix

By applying a custom matrix you can translate, scale and rotate items in one step

Parameter

  • values: of the 3x3 matrix (dx and dy are m13 and m23 values of the matrix)

Return value

  • item instance

item.setMatrix ( m11, m12, m21, m22, dx, dy )

Set the matrix of the canvas element for this item

Current matrix will be overwritten

By applying a custom matrix you can translate, scale and rotate items in one step

Parameter

  • values: of the 3x3 matrix (dx and dy are m13 and m23 values of the matrix)

Return value

  • item instance

itemrotateBy ( angle )

Rotate item by an angle

Parameter

  • angle: rotation angle in degree

Return value

  • item instance

item.rotate ( angle )

Set rotation of item to an angle

Parameter

  • angle: rotation angle in degree

Return value

  • item instance

item.scaleBy ( dx, dy )

Scale item by given value

0.1 would be +10% size

Parameter

  • dx: scale values for x axis
  • dy: scale values for y axis

Return value

  • item instance

item.scale ( x, y )

Scale item to given facor

0.1 would be 10% of original size

Parameter

  • x: scale values for x axis
  • y: scale values for y axis

Return value

  • item instance

item.moveBy ( dx, dy )

Move item by given pixels

Parameter

  • dx: pixel offset for x axis
  • dy: pixel offset for y axis

Return value

  • item instance

item.move ( x, y )

Move item to given coordinates

Parameter

  • dx: pixel offset for x axis
  • dy: pixel offset for y axis

Return value

  • item instance

item.setDrawOptions ( options )

Set draw options for the item

Parameter

  • options: literal object of the options

Return value

  • item instance

item.draw ( canvas, options )

Draw routine for the items

Each item is drawn as path to the canvas

Parameter

  • canvas: cajal instance
  • options: draw options for this draw call

cajal.Circle ( x, y, r )

Create a circle

Parameter

  • x: position in pixel
  • y: position in pixel
  • r: radius of the circle

Return value

  • circle item instance

cajal.Rect ( x, y, w, h, r )

Create a rectangle

Optional with rounded corners if radius is set

Parameter

  • x: position on the x axis in pixel
  • y: position on the y axis in pixel
  • w: height of the rectangle in pixel
  • h: width of the rectangle in pixel
  • r: (optional) radius for rounded corners

Return value

  • rectangle item instance

cajal.Path ( x, y )

Create a path

Parameter

  • x: start of the path at position x in pixel
  • y: start of the path at position y in pixel

Return value

  • path item instance

cajal.line ( x, y )

Create a line to (x,y)

Parameter

  • x: position of the end point on the x-axis in pixel
  • y: position of the end point on the y-axis in pixel

Return value

  • path item instance

path.to ( x, y )

Create a line to (x,y)

Synonym for line (x, y)

path.close ( )

Close the path

Return value

  • path item instance

path.quadraticCurve ( x, y, cx, cy )

Create a quadratic bezier curve to (x,y) with the control point (cx,cy)

Parameter

  • x: position of the end point on the x-axis in pixel
  • y: position of the end point on the y-axis in pixel
  • cx: position of the control point on the x-axis in pixel
  • cy: position of the control point on the y-axis in pixel

Return value

  • path item instance

path.bezierCurve ( x, y, c1x, c1y, c2x, c2y )

Create a bezier curve to (x,y) with the control points (c1x,c1y) and (c2x,c2y)

Parameter

  • x: position of the end point on the x-axis in pixel
  • y: position of the end point on the y-axis in pixel
  • c1x: position of the first control point on the x-axis in pixel
  • c1y: position of the first control point on the y-axis in pixel
  • c2x: position of the second control point on the x-axis in pixel
  • c2y: position of the second control point on the y-axis in pixel

Return value

  • path item instance

item.center ( )

Get the (approximate) center of the item for rotation

Return value

  • point object of the center

Return value

Parameter

Return value
